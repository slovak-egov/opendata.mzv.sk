Patch dataset

Pomocou package_patch môžete čiastočne aktualizovať a/alebo vytvárať zdroje. Ak aktualizujete existujúce zdroje, nezabudnite zadať id zdroja. Existujúce zdroje vylúčené z package_patch data_dict budú odstránené. Zdroje v balíku data_dict bez id sa budú považovať za nové zdroje a pridajú sa. Nové zdroje pridané metódou patch nevytvárajú predvolené zobrazenia.

ckan.logic.action.patch.package_patch(context, data_dict) 

curl -X POST {{ h.url_for(controller='api', action='action', ver=3, logic_function='package_patch', qualified=True) }} \
-H "Authorization: {API_KEY}" \
-d '{

   }'
