Update dataset

Metódy update môžu odstrániť parametre, ktoré nie sú výslovne uvedené v data_dict. Ak chcete upraviť iba konkrétny atribút, použite namiesto toho package_patch.

Odporúča sa zavolať ckan.logic.action.get.package_show(), vykonať požadované zmeny vo výsledku a potom s ním zavolať package_update().

ckan.logic.action.update.package_update(context, data_dict) 

curl -X POST {{ h.url_for(controller='api', action='action', ver=3, logic_function='package_patch', qualified=True) }} \
-H "Authorization: {API_KEY}" \
-d '{
   ...
   "notes": "Update popisky datasetu",
   "owner_org": "nazov_organizacie",
   "license_id": "cc-zero",
   ...
   }'

Odpoveď:

{
  "help": "{{ h.url_for(controller='api', action='action', ver=3, logic_function='help_show', qualified=True) }}?name=package_patch",
  "success": true,
  "result": {
    "license_title": "Creative Commons CCZero",
    ...
  }
}
