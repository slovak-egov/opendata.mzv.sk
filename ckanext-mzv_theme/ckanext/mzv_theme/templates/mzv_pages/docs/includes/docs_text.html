Ckan v{{ h.ckan_version() }}

CKAN je kompletné softvérové riešenie, vďaka ktorému sú údaje prístupné a použiteľné – poskytovaním nástrojov na zefektívnenie publikovania, zdieľania, vyhľadávania a používania údajov (vrátane ukladania údajov a poskytovania robustných dátových rozhraní API). CKAN je zameraný na vydavateľov údajov (národné a regionálne vlády, spoločnosti a organizácie), ktorí chcú sprístupniť svoje údaje.

Viac informácií (všetky rozhrania) k CKAN Data API sa nachádza na oficiálnej stránke v sekcii API sprievodca.

Niekoľko užitočných API koncových bodov

CKAN Action API je výkonné API v RPC-štýle, ktoré vystavuje všetky základné CKAN funkcie API klientom. Všetky základné funkcie webovej stránky CKAN (všetko, čo môžete robiť s webovým rozhraním) môže byť použité externým kódom, ktorý volá CKAN API.

Curl

Syntax príkazu curl je nasledovná:

curl [options ] [URL... ]

Tu sú možnosti, ktoré môžme použiť pri volaní:

  • -X, --request - Metóda HTTP, ktorá sa má použiť.
  • -i, --include - Zahrnie hlavičky odpovedí.
  • -d, --data - Údaje, ktoré sa majú odoslať.
  • -H, --header - Ďalšia hlavička, ktorá sa má odoslať.

Rozdiel medzi metódami update a patch

Rozdiel medzi metódami update a patch spočíva v tom, že patch vykoná aktualizáciu poskytnutých parametrov, pričom všetky ostatné parametre ponechá nezmenené, zatiaľ čo metódy update odstránia všetky parametre, ktoré nie sú výslovne uvedené v data_dict.
